When using Ruby there are lots of tools that one needs to learn. But, perhaps, one of the most important sets of tools to learn are Enumerables. Enumerables are methods that can be used on a collection of data like an array or a hash. Enumerables can allow you to search for specific values, sort by specific values or select specific values.

One very useful method I have discovered is the map enumerable. Map allows you to pull all of the items in an array and change them one by one based on the rules of you code.
