Iteration Cheat Sheet

Iteration In Ruby

When wanting to iterate over an item in ruby, there are a number of built in tools that you can utilize.

  • You can use the each do function. Written out as either "object.each do |x| block" or "object.each { |x| block}"
  • For hashes make sure to include two variables in your code so you can access keys and values i.e. "object.each { |x, y| block}"
  • If you have a set amount of times you want to do something you can use the "x.times do block" or "x.times{block}".
  • When manipulating arrays, remember pop and push can often be helpful items in completing tasks.
  • If you are changing variables or objects make sure to consider whether your method is destructive or not.
  • For more info and objects checkout Ruby Docs

Iteration in JavaScript

When wanting to iterate over an item in Javascript, there is less flexibility than Ruby. You will generally need to use a for loop to iterate in JavaScript

  • When iterating over an array you will want to use a for loop that looks like this for(i=0; i ‹ array.length; i++) {block}. Use array[i] to call on the location you need to iterate on
  • When iterating over an object in JavaScript the for loop will look like this for(var key in object) {block}. You can then call the item you need by using object[key].
  • Remember you do not have to use key, it can be any variable. This can be important if you want to nest for loops within each other to access nested objects.
  • For more info check out Mozilla Developer Network

Remember these are the basic tools to get the job done. Be creative, there are lots of solutions to most problems. Experiment and have fun to create the right solution for your task.
